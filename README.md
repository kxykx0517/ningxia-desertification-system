# 宁夏荒漠化智能治理系统

## 项目简介
针对宁夏荒漠化治理中监测手段落后、方案粗放的问题，构建“感知-决策-执行-反馈”闭环的智能治理系统。

## 技术栈
- Python / Flask（后端API）
- Neo4j（知识图谱）
- ECharts（数据可视化）
- HTML/CSS/JS（前端）

## 核心功能
- **知识图谱构建**：基于Neo4j设计“沙区-沙化类型-严重程度-治理技术”7类标签体系，建立语义关联
- **智能推荐引擎**：三级匹配策略（精确→语义拆分→优先级排序），解决复合沙化类型精准匹配难题
- **可视化看板**：沙化分布地图、数据分析、智能推荐页面

## 项目成果
- 覆盖宁夏6个典型沙区（中卫沙坡头、灵武白芨滩、盐池县等）
- 获学院创新计划立项支持
- 计划申请软件著作权
